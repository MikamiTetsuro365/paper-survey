# Paper survey

* 読もうと思っている論文はIssueに上げる。
* ProjectsのRelated Worksに論文を読んだかどうかを管理する。
* [format.md](https://github.com/shunk031/paper-survey/blob/master/format.md)にしたがって論文の要旨をまとめる。
